How does this work?

1. This password generator lets you turn a text file and a number of your choice into a unique password. You can customise the length of this password, and you have the option to add an image to the generator that will randomise it further.

2. The password generated relies on these inputs, so it will always be the same if the inputs are the same. This allows you to create long and complex passwords while only having to remember a small number of input variables.
